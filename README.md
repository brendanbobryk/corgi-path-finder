A small React game where you help a corgi find a path to its treat by placing obstacles on a grid.
This project focuses on practicing grid-based logic, breadth-first search (BFS), and interactive state updates in React.

## Features

- Click tiles to place or remove obstacles
- Visual grid-based layout
- Breadth-first search pathfinding
- Highlights the shortest valid path
- Dark-themed, simple UI

## Technologies Used

- React (with hooks)
- JavaScript
- CSS
- Vite

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/brendanbobryk/corgi-path-finder.git

2. Navigate to the project folder:
    ```bash
    cd corgi-path-finder

3. Install dependencies:
    ```bash
    npm install

4. Start the development server:
    ```bash
    npm run dev

5. Open your browser at the local address shown in the terminal (usually http://localhost:5173) to play!
